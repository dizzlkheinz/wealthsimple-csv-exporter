# Wealthsimple CSV Exporter

![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)

A simple bookmarklet to export your Wealthsimple Cash transaction history to CSV format, compatible with budgeting tools like YNAB, Excel, and Google Sheets.

## Features

- **One-Click Export**: Export transactions directly from your Wealthsimple activity page
- **CSV Format**: Date, Payee, Amount columns in a clean CSV format
- **YNAB-Compatible**: Dates formatted as YYYY-MM-DD for easy import
- **Smart Date Handling**: Automatically converts "Today", "Yesterday", and relative dates
- **Skips Pending**: Only exports completed transactions
- **Clean Currency Parsing**: Handles various negative sign symbols (âˆ’, â€“, â€”)
- **No Duplicates**: Intelligent deduplication prevents duplicate entries
- **Privacy-Friendly**: Runs entirely in your browserâ€”no data sent anywhere

## Usage

1. Log in to [Wealthsimple Cash](https://my.wealthsimple.com/)
2. Navigate to your Activity/Transactions page
3. **Important**: Scroll down to load all the transactions you want to export (Wealthsimple loads them dynamically)
4. Click the bookmarklet in your bookmarks bar
5. A CSV file will automatically download

## CSV Output Format

The exported CSV contains three columns:

```csv
Date,Payee,Amount
2025-11-22,"Starbucks",-5.67
2025-11-21,"Direct Deposit - Employer",2500.00
2025-11-20,"Transfer to Savings",-500.00
```

- **Date**: YYYY-MM-DD format (sorts correctly in Excel/Sheets)
- **Payee**: Transaction description (quoted for CSV safety)
- **Amount**: Positive for income, negative for expenses

## Importing to YNAB

1. Export your Wealthsimple transactions using the bookmarklet
2. In YNAB, go to your account â†’ Import Transactions
3. Select the downloaded `wealthsimple_activity.csv` file
4. YNAB will automatically match the Date, Payee, and Amount columns

## Troubleshooting

### No transactions exported?

- Make sure you're on the Activity/Transactions page
- Scroll down to load more transactions (they load as you scroll)
- Check if transactions are marked as "Pending" (they're skipped)

### Bookmarklet doesn't work?

- Some browsers block javascript: bookmarklets by default
- Try creating the bookmark manually (see Method 2)
- Make sure you copied the entire code including `javascript:` at the start

### Dates look wrong?

- Dates without years (e.g., "Nov 22") assume the current year
- "Today" and "Yesterday" are converted to actual dates
- All dates are output as YYYY-MM-DD

## Privacy & Security

- All code runs entirely in your browser
- No data is sent to any external server
- The bookmarklet only reads visible transaction data
- Source code is fully visible in `wealthsimple_csv_export_readable.js`
